## ⚡ **緊急対応プラン**

### 🚨 **システム障害時**
1. **サーバー停止**: プロセス確認・再起動
   ```bash
   netstat -ano | findstr :8002
   taskkill /F /PID [PID番号]
   ```

2. **パンドラ機能異常**: デバッグモード起動
   ```bash
   # ログレベル DEBUG で起動
   export LOG_LEVEL=DEBUG
   python src/phase3_ui_bridge_server_pandora.py
   ```

3. **設定ファイル破損**: バックアップからリストア
   ```bash
   cp backup/latest_pandora.yaml personas/pandora.yaml
   ```

### 🔧 **軽微な問題対応**
- **WebSocket接続エラー**: ブラウザリロード・キャッシュクリア
- **API応答遅延**: プロセス監視・リソース確認
- **UI表示異常**: ブラウザ開発者ツールでのデバッグ
