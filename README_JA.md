## 🚀 クイックスタート

### **1. リポジトリのクローン**
```bash
git clone https://github.com/pepepepepepo/sajinos.git
cd sajinos
```

### **2. 環境セットアップ**
```bash
# 開発環境に移動
cd .venv

# 依存関係インストール（必要に応じて）
pip install -r requirements.txt
```

### **3. Ultimate Creative Studio 起動**
```bash
# メインアプリケーション起動
python creative_studio_multimodel_dashboard.py

# アクセス: http://localhost:8017
```

### **4. ワークスペース探索**
- **チャット**: http://localhost:8017/chat-workspace
- **開発**: http://localhost:8017/development-workspace
- **音楽**: http://localhost:8017/music-workspace
- **分析**: http://localhost:8017/analysis-workspace
- **コントロール**: http://localhost:8017/control-workspace

- **フロントエンド**: HTML5 + CSS3 + JavaScript (ES6+)
